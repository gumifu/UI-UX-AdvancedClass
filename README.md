# UIUX Class Demo

Interactive scroll demo with Apple-style controller.

## Features

- Smooth scrolling with custom controller
- Apple-style glassmorphism design
- Carousel indicator with 5 dots
- Long-press to activate scroll control
- Mobile-optimized (carousel pauses on mobile)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

- **Long-press the dots** on the right side to activate scroll control
- **Drag** to scroll up/down
- The carousel automatically scrolls through sections (paused on mobile)
